# bt_api_mercado_bitcoin

Mercado Bitcoin exchange plugin for bt_api framework.

## Installation

```bash
pip install bt_api_mercado_bitcoin
```

## Usage

```python
from bt_api_mercado_bitcoin import MercadoBitcoinExchangeDataSpot, MercadoBitcoinRequestDataSpot
```

## Features

- REST API support
- HMAC-SHA512 authentication
- Symbol format: `BTC` (uses BRL pairs internally)
